kAPI_OPT_NO_RESP = ':@no-response' (line 83)

No response option.

Can be a boolean or integer, defaults to FALSE. If TRUE, the response section will not be included in the result. This can be useful when you are only interested in the status of the operation and not in the response.

kAPI_OP_GET_OBJECT_REF = '@GET-REF' (line 47)

GET-OBJECT-REF web-service.

This tag defines a web service that returns an object by reference. It is equivalent to the kAPI_OP_GET_ONE operation, except that instead of using the query provided in the kAPI_DATA_QUERY parameter, it will try to extract an identifier from the object provided in the kAPI_DATA_OBJECT parameter.

Note that as with other values in the object parameter, you must serialise the value.
