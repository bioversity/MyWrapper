CMeasureTerm

Description
Description | Vars | Methods (details) Constants

Scale term.

This kind of term represents a measure term, a term that defines a measurement scale which refers to a unit term and features a data type; such terms can be used to annotate data elements.

These two properties are managed respectively by the Unit and Type methods, the type property must be present for object's status to be initialised.

The class also enforces the kTYPE_MEASURE_TERM kind.

Manage valid reference.

This method can be used to handle the measure unit, it uses the standard accessor method to manage the offset.

The value provided to this property should be the identifier of a term hat defines a unit.

For a more in-depth reference of this method, please consult the _ManageOffset method, in which the first parameter will be the constant kTAG_VALID.

In this class we feed the value to the_CheckReference method that will take care of handling object references.
