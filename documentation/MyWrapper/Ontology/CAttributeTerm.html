CAttributeTerm

Description | Vars | Methods (details) Constants

Attribute term.

This kind of term represents an attribute term. In general, it represents a term that is mainly used to qas an attribute of other terms.

In this class we enforce the kTAG_TERM_ATTRIBUTE kind.
