FAO/WIEWS institute.

This class overloads its ancestor to implement an institute entity taken from the FAO WIEWS institutes database.

This kind of institutes adds the following properties:

  • kENTITY_INST_FAO_EPACRONYM: This offset represents the institute ECPGR acronym.
  • kENTITY_INST_FAO_TYPE: This offset represents the institute set of FAO institute types.
  • kOFFSET_URL: This offset represents the institute URL.
  • kOFFSET_LATITUDE: This offset represents the institute latitude, note this is an integer value.
  • kOFFSET_LONGITUDE: This offset represents the institute longitude, note this is an integer value.
  • kOFFSET_ALTITUDE: This offset represents the institute altitude, note this is an integer value.

The object unique identifier is formed by using the code without any formatting.

static Update (line 289)

Update institutes.

This method will download the current FAO/WIEWS export file and load its contents into the provided container.

If you provide the second parameter, it will be used to filter only institutes modified after the provided date; the format of that parameter is a date as YYYY-MM-DD.

The method will return the number of updated institutes.
