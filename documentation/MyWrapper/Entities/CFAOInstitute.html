FAO/WIEWS institute.

This class overloads its ancestor to implement an institute entity taken from the FAO WIEWS institutes database.

This kind of institutes adds the following properties:

  • kENTITY_INST_FAO_EPACRONYM: This offset represents the institute ECPGR acronym.
  • kENTITY_INST_FAO_TYPE: This offset represents the institute set of FAOType() institute types.
  • kOFFSET_URL: This offset represents the institute URL.
  • kOFFSET_LATITUDE: This offset represents the institute latitude, note this is an integer value.
  • kOFFSET_LONGITUDE: This offset represents the institute longitude, note this is an integer value.
  • kOFFSET_ALTITUDE: This offset represents the institute altitude, note this is an integer value.

The object unique identifier is formed by using the code without any formatting.

URL (line 200)

Manage institute urls.

This method can be used to manage the institute URL or web pages list, the method expects the following parameters:

  • kTAG_KIND: The institute URL kind, this could be main, sales or international. This element represents the array key, although technically it is implemented as an element to allow searching on all values.
  • kTAG_DATA: The URL or web page address.

The parameters to this method are:

  • $theValue: The value or operation:
    • NULL: Return the current value selected by the second parameter.
    • FALSE: Delete the value selected by the second parameter.
    • other: Set value selected by the second parameter.
  • $theType: The element type, kind or index:
    • NULL: This value indicates that the URL has no type or kind, in general, when adding elements, this case applies to default elements.
    • other: All other types will be interpreted as the kind or type of the URL.
  • $getOld: Determines what the method will return:
    • TRUE: Return the value before it was eventually modified.
    • FALSE: Return the value after it was eventually modified.

  • access: public
