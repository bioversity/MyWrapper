kTYPE_REF = ':REF' (line 138)

Object reference type.

This data type should be used as object references, it is generally expressed as a structure composed of the following elements:

  • kTAG_REFERENCE_ID: Object local unique identifier. This value may be used as a scalar in cases in which the location of the object is univoque.
  • kTAG_REFERENCE_CONTAINER: Object container, reference to the container in which the object is stored. This component will be used in cases in which the container of the object is not univoque.
  • kTAG_REFERENCE_DATABASE: Database container, reference to the database or container superclass in which the object is stored. This component will be used in cases in which the database of the object is not univoque.
