kDATA_TYPE_BINARY = ':BIN' (line 138)

Binary type.

This value represents the primitive binary data type, it is assumed that it is provided as a hexadecimal string.

This data type is serialised as follows:

  • kTAG_TYPE: Will contain this constant.
  • kTAG_DATA: Will contain the following structure:
    • kOBJ_TYPE_BINARY_BIN: The binary string.
    • kOBJ_TYPE_BINARY_TYPE: Binary string type (integer):
      • 1: Function.
      • 2: Byte array (use as default).
      • 3: UUID.
      • 5: MD5.
      • 128: Custom.
