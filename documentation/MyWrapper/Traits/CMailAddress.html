Mailing address.

This class implements a mailing address, it wraps the parent class around a structure that defines a mailing address.

In general instances of this class will be embedded in persistent objects to add lists of addresses, the current class does not feature any persistence functions, it only concentrates in managing an address as a set of properties:

  • Place: This property defines a named place or location, it should be used only if required.
  • Care of: This property indicates who is the owner or reference at the address that is not the same as the sender. It should be used only if required.
  • Street: This property indicates the street name or P.O. box number.
  • Zip: This property indicates the ZIP code.
  • City: This property indicates the address city name.
  • Province: This property indicates the address province name or code.
  • Country: This property indicates the address country name or code.
  • Full address: This property can be used when an address does not have its properties separated, or it can be used as an export feature.
