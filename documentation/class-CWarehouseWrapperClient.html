Class CWarehouseWrapperClient

Warehouse data wrapper client.

This class represents a germplasm warehouse web-services data wrapper client, it facilitates the job of requesting information from servers derived from the CWarehouseWrapper CWarehouseWrapper class.

This class adds the following properties to its CWrapperClient ancestor:

  • User UserCode() code: This kAPI_OPT_USER_CODE property represents the user code provided to the kAPI_OP_LOGIN login operation.
  • User UserPass() password: This kAPI_OPT_USER_PASS property represents the user password provided to the kAPI_OP_LOGIN login operation.

The class also adds the following new operations:

  • kAPI_OP_LOGIN kAPI_OP_LOGIN: This is the tag that represents the login operation, it will return the matching user CUser object if the provided user kAPI_OPT_USER_CODE code and kAPI_OPT_USER_PASS password match.
