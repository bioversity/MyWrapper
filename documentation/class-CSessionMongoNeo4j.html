Class CSessionMongoNeo4j

Mongo and Neo4j session object.

This concrete class implements a session that uses MongoDB as its data DataStore() store and Neo4j as its GraphStore() store.

The class does not add any extra functionality to its CSessionObject parent, except that this class can be instantiated.
