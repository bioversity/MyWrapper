Class CMailAddress

Mailing address.

This class implements a mailing address, it wraps the CArrayObject parent class around a structure that defines a mailing address.

In general instances of this class will be embedded in CPersistentObject persistent objects to add lists of addresses, the current class does not feature any persistence functions, it only concentrates in managing an address as a set of properties:

  • Place() Place: This kOFFSET_PLACE property defines a named place or location, it should be used only if required.
  • Care() Care of: This kOFFSET_CARE property indicates who is the owner or reference at the address that is not the same as the sender. It should be used only if required.
  • Street() Street: This kOFFSET_STREET property indicates the street name or P.O. box number.
  • Zip() Zip: This kOFFSET_ZIP_CODE property indicates the ZIP code.
  • City() City: This kOFFSET_CITY property indicates the address city name.
  • Province() Province: This kOFFSET_PROVINCE property indicates the address province name or code.
  • Country() Country: This kOFFSET_COUNTRY property indicates the address country name or code.
  • Full() Full address: This kOFFSET_FULL property can be used when an address does not have its properties separated, or it can be used as an export feature.

Manage place.

Manage place.

This method can be used to manage a mailing address kOFFSET_PLACE place, it accepts a parameter which represents either the place name or the requested operation, depending on its value:

  • NULL: Return the current value.
  • FALSE: Delete the current value.
  • other: Set the value with the provided parameter.

The second parameter is a boolean which if TRUE will return the old value when replacing values; if FALSE, it will return the currently set value.
