Class CGraphEdge

Graph edge.

This class implements a graph edge.

This class extends its CGraphNode parent class in that its Node() node property is a relationship node rather than a plain node.

# Type( mixed $theValue = NULL, boolean $getOld = FALSE )

Manage node type.

Manage node type.

This property corresponds to the node type, or predicate reference, this property is a string, and it is managed directly by the relationship node: we simply wrap this method around the native management:

  • $theValue: The value or operation:
    • NULL: Return the current value.
    • other: Set the value converted to string.
  • $getOld: Determines what the method will return:
    • TRUE: Return the value before it was eventually modified.
    • FALSE: Return the value after it was eventually modified.
