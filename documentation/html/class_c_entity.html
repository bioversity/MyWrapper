Manage entity mailing addresses.

This method can be used to manage the entity mailing addresses, it uses the standard accessor method to manage the list of addresses.

This list is an array of structures where the kTAG_TYPE element indicates the type of the address, for instance home or office, and the kTAG_DATA element represents the actual address, be it a string or itself a structure holding the address elements. If the kTAG_TYPE element is missing, we assume it to be the default address.

For a more in-depth reference of this method, please consult the _ManageTypedArrayOffset method, in which the first parameter will be the constant kOFFSET_MAIL.

Note that this method will <i<NOT work on an object that was serialised.

CEntity::Parent ( $  theValue,
$  theOperation = NULL,
$  getOld = FALSE 
)

Manage entity parents.

This method can be used to manage the entity parents, you can add/replace, retrieve and delete parent elements depending on the value of the parameters.

The property is represented by an array of instances derived from this class, or strings representing entity identifiers. In the first case, prior to saving the current object, array elements in the form of CEntity instances will also be saved and replaced by their identifiers.

This method accepts the following parameters:

  • $theIndex: This parameter represents the element index.
  • $theOperation: The operation to perform:
    • NULL: Retrieve the element, the methoid will return the matching element or NULL if not found.
    • FALSE: Delete the element, depending on the value of the next parameter, the method will either return the deleted element or NULL.
    • other: Any other value means that we want to add/replace the element, in this case the method will return the added element's index.
  • $theData: This parameter represents the element data.

Note that this method will <i<NOT work on an object that was serialised.

This method can be used to manage the entity types, it uses the standard accessor method to manage the list of parent entities.
