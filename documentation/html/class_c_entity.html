CEntity::Email (   $theValue = NULL,

)

Manage entity e-mail.

This method can be used to manage the entity e-mail, it uses the standard accessor method to manage the offset:

  • $theValue: The value or operation:
    • NULL: Return the current value.
    • FALSE: Delete the value.
    • other: Set value.
  • $getOld: Determines what the method will return:
    • TRUE: Return the value before it was eventually modified.
    • FALSE: Return the value after it was eventually modified.

CEntity::Name (   $theValue = NULL,
  $getOld = FALSE 
)

Manage entity name.

This method can be used to handle the entity name, it uses the standard accessor method to manage the offset.

This value should be a string that can be used as a label or as a short definition of the entity. The name may be language dependent, so the type of data stored in this offset is the responsibility of concrete classes.

For a more in-depth reference of this method, please consult the CAttribute::ManageOffset method, in which the second parameter will be the constant kTAG_NAME.
