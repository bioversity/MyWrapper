CDataTypeInt64::__construct (   $theData = NULL)

Instantiate class.

In this class we store kTYPE_INT64 in the kTAG_TYPE offset and the integer in the kTAG_DATA offset.

The method will check if the provided data (converted to string) is numeric, if this is not the case, it will raise an exception. This is to handle miscellaneous objects.

The number will be stored in the kTAG_DATA offset as an integer, if the current system is 64 bits.
