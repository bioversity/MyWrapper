CWarehouseWrapperClient::Attributes (   $theKey,
  $theValue = NULL,
  $theOperation = NULL,
  $getOld = FALSE 
)

Manage selectors list.

This method can be used to manage the attribute selectors, it manages the following array structure:

  • Key: The array key should correspond to the node attribute you want to match:
    • kTAG_TYPE: The ontology node type.
    • kTAG_KIND: The ontology node kind.
    • kTAG_DOMAIN: The domain of the ontology.
    • kTAG_CATEGORY: The category of the ontology.
  • Value: An array of values to be matched.

The elements of these key/value pairs will be compared in AND.

The parameters to this method are:

  • $theKey: The attribute key.
  • $theValue: The attribute value:
    • NULL: Apply operation to all values.
    • other: The attribute value.
  • $theOperation: The operation:
    • NULL: Return the attribute value matched by the first and second parameters, or return all values matching the first parameter if the second is NULL.
    • FALSE: Delete the attribute value matched by the first and second parameters, or delete all values matching the first parameter if the second is NULL.
    • TRUE: Add the value in the second parameter to the attributes matching the first parameter, or replace all values matching the first parameter with the value in the second parameter.

Manage edges direction.

This method can be used to manage the edges direction, it accepts a string which represents either the relationship direction, or the requested operation:

  • NULL: Return the current value.
  • FALSE: Delete the current value.
  • other: Set the value with the provided parameter:
    • kAPI_DIRECTION_IN: Incoming relationships.
    • kAPI_DIRECTION_OUT: Outgoing relationships.
    • kAPI_DIRECTION_ALL: Both incoming and outgoing relationships.

The second parameter is a boolean which if TRUE will return the old value when replacing values; if FALSE, it will return the currently set value.
