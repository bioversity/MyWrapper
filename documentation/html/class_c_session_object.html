CSessionObject::_InitUser (   $theOperation)
protected

Initialise user.

This method will initialise the current user object, the method expects one boolean parameter: TRUE means that the user is to be set, FALSE means that the user is to be reset.

Mote that this method is only called when sleeping or waking: it will save eventual changes made to the user when sleeping and reload the user when waking.

The method returns no result, any error should trigger an exception.

In this class we commit user data and store the user identifier when going to sleep and load the user back when waking.

Parameters
boolean$theOperationTRUE set, FALSE reset.

CSessionObject::Database (   $theValue = NULL,
  $getOld = FALSE 
)

Manage the database reference.

This method can be used to manage the session's default database reference. This database represents the connection to the default collections container.

The method accepts two parameters:

  • $theValue: The value to set or the operation to perform:
    • NULL: Retrieve the current value.
    • FALSE: Delete the current value, it will be set to NULL.
    • other: Any other value will be interpreted as the new value to be set.
  • $getOld: A boolean flag determining which value the method will return:
    • TRUE: Return the value before it has eventually been modified, this option is only relevant when deleting or relacing a value.
    • FALSE: Return the value after it has eventually been modified.
Parameters
mixed$theValueValue or operation.
boolean$getOldTRUE get old value.

CSessionObject::Login (   $theCode,
  $thePass 
)

Perform a login.

This method can be used to query a user by code and password.

The method will look for the user in the default users container, if found, it will return the object, if not, it will return NULL.

The method expects two parameters:

  • $theCode: The user code.
  • $thePass: The user password.

In this class we declare the method abstract, because the specifics of seeking for the user depends on the specific database engine.

Parameters
string$theCodeUser code.
string$thePassUser password.
