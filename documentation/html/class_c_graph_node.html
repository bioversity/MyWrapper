CGraphNode::Node (   $theValue = NULL,
  $getOld = FALSE 
)

Manage native node.

This method can be used to manage the native node reference, it uses the standard accessor method to manage the property:

  • $theValue: The value or operation:
    • NULL: Return the current value.
    • FALSE: Delete the value.
    • Everyman: Set value.
    • other: Raise exception.
  • $getOld: Determines what the method will return:
    • TRUE: Return the value before it was eventually modified.
    • FALSE: Return the value after it was eventually modified.

The method will also set the dirty status and the inited status if the node is provided.

Parameters
mixed$theValueNode or operation.
boolean$getOldTRUE get old value.

CGraphNode::RelateTo (   $theContainer,
  $thePredicate,
  $theObject 
)

Relate node.

This method can be used to relate the current node to anther node, it will return a Everyman instance.

The parameters to this method are:

  • $theContainer: The graph container.
  • $thePredicate: The predicate string or edge type.
  • $theObject: The destination node or relationship object node.

The method will return a Everyman object, or raise an exception if the operation was not successful.

This method follows the logic of the relateTo() method of Everyman.

mixed$theContainerGraph container.
string$thePredicatePredicate.
