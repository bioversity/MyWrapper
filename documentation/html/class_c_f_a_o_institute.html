Manage FAO/WIEWS types.

This method can be used to handle the institute FAO/WIEWS types list, it uses the standard accessor method to manage the list of acronyms.

Each element of this list should indicate an acronym by which one refers to the current institute, the nature and specifics of these elements is the responsibility of concrete classes.

static CFAOInstitute::Import (   $theContainer,
  $theURL = kENTITY_INST_FAO_DOWNLOAD 
)
static

Import institutes.

This method can be used to import the institutes list from the current FAO/WIEWS export file, the method expects two parameters:

  • $theContainer: The container in which the institutes are stored.
  • $theURL: The download URL of the FAO/WIEWS export file.

The method will return the number of records added and replaced.

Note that the method will commit the records using the kFLAG_PERSIST_REPLACE and kFLAG_STATE_ENCODED flags.

CContainer$theContainerData container.
