 37:  * Entity.
 38:  *
 39:  * An entity can be an individual, and organisation or a legal entity that exists as a unit,
 40:  * rather than being embedded in another object.
 41:  *
 42:  * In this class we add to the {@link CCodedUnitObject parent} the following properties:
 43:  *
 44:  * <ul>
 45:  *  <li><i>{@link kTAG_NAME kTAG_NAME}</i>: This offset represents the entity name, the
 46:  *      class features a member accessor {@link Name() method} to manage this property.
 47:  *  <li><i>{@link kOFFSET_EMAIL kOFFSET_EMAIL}</i>: This offset represents the entity e-mail
 48:  *      address, it is a scalar property and the class features a member accessor
 49:  *      {@link Email() method} to manage it.
 50:  * </ul>
 51:  *
 52:  * The class also features a static {@link DefaultContainer() method} that returns the
 53:  * default container name for objects of this type.
 54:  *
 55:  * Objects derived from this class will have their {@link _IsEncoded() encoded}
 56:  * {@link kFLAG_STATE_ENCODED flag} set by default and must implement a protected
 57:  * {@link _TokeniseIdentifier() method} that must add tokens to the object
 58:  * {@link _index() index} in order to ensure a unique identifier.
