 43: /**
 44:  * Institute.
 45:  *
 46:  * This class overloads its {@link CContact ancestor} to implement an institute contact
 47:  * entity.
 48:  *
 49:  * Institutes feature specific properties that are added to the inherited ones:
 50:  *
 51:  * <ul>
 52:  *  <li><i>{@link kOFFSET_ACRONYM kOFFSET_ACRONYM}</i>: This offset represents the institute
 53:  *      list of {@link Acronym() acronyms}.
 54:  *  <li><i>{@link kOFFSET_URL kOFFSET_URL}</i>: This offset represents the institute list of
 55:  *      URLs or web pages.
 56:  * </ul>
 57:  *
 58:  * The object is considered {@link _IsInited() initialised} only if it has its
 59:  * {@link Code() code}, as its {@link CEntity ancestor}, its {@link Name() name},
 60:  * {@link Password() password} and {@link Email() e-mail} address.
 61:  *
 62:  * If the {@link Code() code} has not been explicitly set, {@link _PrepareCommit() before}
 63:  * {@link Commit() committing} the object it will be set to the value of the
 64:  * {@link Email e-mail}. Also in that phase, the {@link kENTITY_USER kENTITY_USER} constant
 65:  * will be set in the user {@link Type() type}.
 66:  *
 67:  * The {@link Email() e-mail} in this class is a scalar property, in other classes it will
 68:  * probably be a list of different e-mail types. In this class we want to link a single
 69:  * user with a single e-mail, possibly not shared by any other user, that is why we link by
 70:  * default the user {@link Code() code} and {@link Email() e-mail}.
 71:  *
 72:  * The unique identifier of this class is composed by the default {@link _index() index} of
 73:  * thew object, prefixed by the {@link kENTITY_INST kENTITY_INST} token and the
 74:  * {@link kTOKEN_CLASS_SEPARATOR kTOKEN_CLASS_SEPARATOR} token, this allows institutes and
 75:  * other types of {@link CEntity entities} to share the same {@link Code() code}; this is
 76:  * enforced both in the {@link HashIndex() HashIndex} method, to which you only need to pass
 77:  * the user {@link Code() code}, and in the protected
 78:  * {@link _PrepareCommit() _PrepareCommit} method which will place the resulting string in
 79:  * the global {@link kTAG_GID identifier}.
 80:  *
 81:  *  @package    MyWrapper
 82:  *  @subpackage Entities
 83:  */
