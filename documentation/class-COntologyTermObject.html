Class COntologyTermObject

Ontology term ancestor.

An ontology term represents a concept identified by a pair of elements: the NS() namespace and the Code() code. The NS() namespace groups all Code() codes belonging to a specific category, the Code() code identifies the term within the NS() namespace.

The unique _index() identifier of instances from this class are formed by the combination of the NS() namespace and @link Code() code}, separated by a kTOKEN_NAMESPACE_SEPARATOR separator token; if the term has no namespace, the token is omitted.

This class represents the ancestor of ontology terms, it adds to its CCodedUnitObject parent the following requirements and features:

  • _CheckReference() _CheckReference: When instantiating the class, the parameter provided as the identifier is required to be an object reference.
  • kFLAG_STATE_ENCODED kFLAG_STATE_ENCODED flag: This flag is enforced, which means that all derived classes must use the standard CDataType types.

We declare this class abstract to force the creation of specific ontology term types.

# _CheckReference( mixed $theReference )

Normalise reference parameter.

Normalise reference parameter.

This method can be used to normalise a parameter that is supposed to be a reference to another term, that is, a binary string hash of the term's _index() identifier converted to a CDataTypeBinary binary standard type.

The method will perform the following conversions:

  • CDataTypeBinary CDataTypeBinary: This is the default data type for the identifier.
  • MongoBinData MongoBinData: This type will be converted to the standard CDataTypeBinary CDataTypeBinary type.
  • COntologyTermObject: Objects of the same class will have their _id() identifier extracted.
  • NULL: NULL data will simply be passed.
  • other: Any other data type is assumed to be the term's _index() identifier, so it will be hashed into a binary string and converted to the standard CDataTypeBinary CDataTypeBinary type.
  • array: Arrays cannot be converted to string, so the method will raise an exception.
