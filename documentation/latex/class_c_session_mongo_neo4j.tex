Serialise object.

This method is called before the object gets \hyperlink{class_c_session_object_a37b9597dba12c0cc62c444e1b371634d}{serialized}, its duty is to normalise the object's properties before these get serialized.

In this class we \hyperlink{class_c_session_mongo_neo4j_ab787e392987093f1209192218bbd90b8}{replace} the users \hyperlink{class_c_session_mongo_neo4j_a705c05fae227d70ca244d018b15a6bd4}{container} with its name, the \hyperlink{class_c_session_mongo_neo4j_ada4d97ca5fb8a3f205517032856a6c1a}{database} \hyperlink{class_c_session_mongo_neo4j_a6311c1e5d89e5de8188472de659bd50d}{with} its name and the graph \hyperlink{class_c_session_mongo_neo4j_aa5e06288fa3a285675da6ba8a2b281d5}{store} \hyperlink{class_c_session_mongo_neo4j_a66633d4e2897ad21fd4bb60365a4eb0a}{with} its transport.

We need to do the above because those objects do not serialize correctly.
