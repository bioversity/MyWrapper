Initialise user.

This method will initialise the current user object, the method expects one boolean parameter\-: {\itshape T\-R\-U\-E} means that the user is to be set, {\itshape F\-A\-L\-S\-E} means that the user is to be reset.

Mote that this method is only called when \hyperlink{}{sleeping} or \hyperlink{}{waking}\-: it will save eventual changes made to the user when \hyperlink{}{sleeping} and reload the user when \hyperlink{}{waking}.

The method returns no result, any error should trigger an exception.

In this class we \hyperlink{class_c_persistent_object_a88b1f2b11d3d60e0b3d33d8b0649b68a}{commit} user data and store the user \hyperlink{}{identifier} when going to \hyperlink{}{sleep} and \hyperlink{class_c_session_object_a2fd936580b9d8f0cecc22755726b8811}{load} the user back when \hyperlink{}{waking}.
