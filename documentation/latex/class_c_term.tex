\item {\itshape \hyperlink{}{k\-T\-A\-G\-\_\-\-D\-A\-T\-A}}\-: This element represents the image data which should be expressed in the data type declared in the \hyperlink{}{k\-T\-A\-G\-\_\-\-T\-Y\-P\-E} element.

\begin{DoxyItemize}
