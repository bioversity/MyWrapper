    Warehouse data wrapper client.

    This class represents a germplasm warehouse web-services data wrapper client, it facilitates the job of requesting information from servers derived from the CWarehouseWrapper class.

    This class adds the following properties to its ancestor:

    • User code: This property represents the user code provided to the login operation.
    • User password: This property represents the user password provided to the login operation.

    The class also adds the following new operations:

    • kAPI_OP_LOGIN: This is the tag that represents the login operation, it will return the matching user object if the provided user code and password match.

    Manage edges direction.

    Direction(integer $theValue, boolean $getOld) : mixed

    This method can be used to manage the edges direction, it accepts a string which represents either the relationship direction, or the requested operation:

    • NULL: Return the current value.
    • FALSE: Delete the current value.
    • other: Set the value with the provided parameter:
      • kAPI_DIRECTION_IN: Incoming relationships.
      • kAPI_DIRECTION_OUT: Outgoing relationships.
      • kAPI_DIRECTION_ALL: Both incoming and outgoing relationships.

    The second parameter is a boolean which if TRUE will return the old value when replacing values; if FALSE, it will return the currently set value.
