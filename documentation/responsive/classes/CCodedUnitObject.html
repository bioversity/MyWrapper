    • \CCodedUnitObject

    Coded unit object.

    Objects derived from this class have the following predefined properties:

    • kTAG_CODE: This attribute represents the current object's code, identifier or acronym.
    • kTAG_KIND: This attribute represents the current object's kind or type.
    • kTAG_CREATED: This offset holds the object's creation time stamp.
    • kTAG_MODIFIED: This offset holds the object's last modification time stamp, this property should be used to mark all objects.

    By default, the unique identifier of the object is its code, which is also its id.

    Objects of this class require at least the code offset to have an initialised status.

    Manage object creation time stamp.

    Created(NULL | FALSE | string $theValue, boolean $getOld) : string

    This method can be used to manage the object creation time-stamp, it uses the standard accessor method to manage the offset:

    • $theValue: The value or operation:
      • NULL: Return the current value.
      • FALSE: Delete the value.
      • other: Set value.
    • $getOld: Determines what the method will return:
      • TRUE: Return the value before it was eventually modified.
      • FALSE: Return the value after it was eventually modified.
