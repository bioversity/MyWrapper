    Coded unit object.

    Objects derived from this class have the code predefined properties, by default, the unique identifier of the object is its code, which is also its id.

    Objects of this class require at least the code offset to have an initialised status.

    package MyWrapper

    Manage code.

    Code(mixed $theValue, boolean $getOld) : string

    This method can be used to handle the object's code, it uses the standard accessor method to manage the offset.

    The code represents the object's identifier.

    For a more in-depth reference of this method, please consult the ManageOffset method, in which the first parameter will be the constant kTAG_CODE.
