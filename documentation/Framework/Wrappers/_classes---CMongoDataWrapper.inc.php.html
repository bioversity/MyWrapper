kAPI_OP_GET_OBJECT_REF = 'GetObjectByReference' (line 45)

GET-OBJECT-REF web-service.

This tag defines a web service that returns an object when provided an object reference: with this option you do not provide a query but you provide a MongoDBRef object in the object parameter.

Note that as with other values in the object parameter, you must serialise the value.
