Class CWrapperClient

Wrapper client.

This class represents a web-services wrapper client, it facilitates the job of requesting information from servers derived from the CWrapper CWrapper class.

This class supports the following properties:

  • Url() URL: This element represents the web-service URL, and it is stored in the kOFFSET_URL kOFFSET_URL offset.
  • Operation() Operation: This element represents the web-service requested operation, it is stored in the kAPI_OPERATION kAPI_OPERATION offset.
  • Format() Format: This element represents the web-service parameters and response format, it is stored in the kAPI_FORMAT kAPI_FORMAT offset.
  • Stamp() Stamp: This element represents a switch to turn on and off timing information from the web service, it is stored in the kAPI_REQ_STAMP kAPI_REQ_STAMP offset.
  • Log LogRequest() request: This element represents a switch to turn on and off logging the request: if on, the request will be returned by the web-service, it is stored in the kAPI_OPT_LOG_REQUEST kAPI_OPT_LOG_REQUEST offset.
  • Log LogTrace() trace: This element represents a switch to turn on and off tracing exceptions: if on, any exception will also hold a trace, it is stored in the kAPI_OPT_LOG_TRACE kAPI_OPT_LOG_TRACE offset.

Objects of this class require at least the Url() URL kOFFSET_URL offset, Operation() operation kAPI_OPERATION offset and the Format() format kAPI_FORMAT offset to be set if they expect to have an _IsInited() initialised kFLAG_STATE_INITED status, which is necessary to send Request() requests to web-services.

Manage format.

This method can be used to manage the kAPI_FORMAT format in which both the parameters are sent and the response is returned from the web-service. It accepts a parameter which represents either the web-service format code or this method operation, depending on its value:

  • NULL: Return the current value.
  • FALSE: Delete the current value.
  • other: Set the value with the provided parameter.

The second parameter is a boolean which if TRUE will return the old value when replacing values; if FALSE, it will return the currently set value.

This method will check whether the provided format is supported, if this is not the case, it will raise an exception. Derived classes should first check their specific operations, if not matched, they should pass the parameter to the parent method. In this class we support the following operations:

  • kTYPE_PHP kTYPE_PHP: Parameters and response will be serialized.
  • kTYPE_XML kTYPE_XML: Parameters and response will be encoded in XML.
  • kTYPE_JSON kTYPE_JSON: Parameters and response will be encoded in JSON.
