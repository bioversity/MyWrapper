Class CArrayObject

Common ancestor.

This class represents the ancestor of most entity mapped classes, it maps objects to an array by deriving from ArrayObject ArrayObject and it defines a couple of general purpose utility static methods.

This class implements the following interfaces:

  • Offsets: In this class there cannot be an offset with a NULL value, the offset itself should be offsetUnset() deleted in that case. Because of this we also override the inherited behaviour by suppressing notices and warnings when offsetGet() getting non-existant offsets.
  • String formatting: We provide a generalised static method to StringNormalise() format strings which accepts a bitfield parameter that indicates which operation to perform, such as kFLAG_MODIFIER_UTF8 UTF8 encode, kFLAG_MODIFIER_LTRIM left and kFLAG_MODIFIER_RTRIM right trim, kFLAG_MODIFIER_NULL NULL handling, kFLAG_MODIFIER_NOCASE case insensitive conversion, kFLAG_MODIFIER_URL URL, kFLAG_MODIFIER_HTML HTML and kFLAG_MODIFIER_HEX HEX encoding and kFLAG_MODIFIER_HASH hashing.
  • Time formatting: We provide a generalised static DurationString() method to display duration strings.
ArrayObject implements IteratorAggregate, Traversable, ArrayAccess, Serializable, Countable
