Class CEntity

Entity.

An entity can be an individual, and organisation or a legal entity that exists as a unit, rather than being embedded in another object.

In this class we add to the CCodedUnitObject parent the following properties:

  • kTAG_NAME kTAG_NAME: This offset represents the entity name, the class features a member accessor Name() method to manage this property.
  • kOFFSET_EMAIL kOFFSET_EMAIL: This offset represents the entity e-mail address, it is a scalar property and the class features a member accessor Email() method to manage it.

The class also features a static DefaultContainer() method that returns the default container name for objects of this type.

Objects derived from this class will have their _IsEncoded() encoded kFLAG_STATE_ENCODED flag set by default and must implement a protected _TokeniseIdentifier() method that must add tokens to the object _index() index in order to ensure a unique identifier.
