Class CCodedUnitObject

Coded unit object.

Objects derived from this class have the Code() code predefined kTAG_CODE properties, by default, the unique _index() identifier of the object is its Code() code, which is also its _id() id.

Objects of this class require at least the Code() code kTAG_CODE offset to have an _IsInited() initialised kFLAG_STATE_INITED status.

Manage code.

This method can be used to handle the object's kTAG_CODE code, it uses the standard accessor CAttribute::ManageOffset() method to manage the kTAG_CODE offset.

The code represents the object's _index() identifier.

For a more in-depth reference of this method, please consult the CAttribute::ManageOffset() ManageOffset method, in which the first parameter will be the constant kTAG_CODE kTAG_CODE.
