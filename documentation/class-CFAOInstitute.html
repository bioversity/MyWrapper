Class CFAOInstitute

FAO/WIEWS institute.

This class overloads its CInstitute ancestor to implement an institute entity taken from the FAO WIEWS institutes database.

This kind of institutes adds the following properties:

  • kENTITY_INST_FAO_EPACRONYM kENTITY_INST_FAO_EPACRONYM: This offset represents the institute ECPGR EAcronym() acronym.
  • kENTITY_INST_FAO_TYPE kENTITY_INST_FAO_TYPE: This offset represents the institute set of FAOType() FAO institute types.
  • kOFFSET_URL kOFFSET_URL: This offset represents the institute URL() URL.
  • kENTITY_INST_FAO_LAT kOFFSET_kENTITY_INST_FAO_LATLATITUDE: This offset represents the institute Latitude() latitude, note this is an integer value.
  • kENTITY_INST_FAO_LON kENTITY_INST_FAO_LON: This offset represents the institute Longitude() longitude, note this is an integer value.
  • kENTITY_INST_FAO_ALT kENTITY_INST_FAO_ALT: This offset represents the institute Altitude() altitude, note this is an integer value.

Unlike its CInstitute parent, this class will use the institute Code() code as-is for the object unique _id() identifier, the index HashIndex() hashing method will not transform the _index() index since it is at most 7 characters thus cannot conflict with other CEntity entity identifiers.

Import institutes.

Import institutes.

This method can be used to import the institutes list from the current FAO/WIEWS export file, the method expects two parameters:

  • $theContainer: The CContainer container in which the institutes are stored.
  • $theURL: The download URL of the FAO/WIEWS export file.

The method will return the number of records added and replaced.

Note that the method will commit the records using the kFLAG_PERSIST_REPLACE kFLAG_PERSIST_REPLACE and kFLAG_STATE_ENCODED kFLAG_STATE_ENCODED flags.
